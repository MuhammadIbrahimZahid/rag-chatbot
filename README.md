# RAG Chatbot Demo (Free AI Version)

A demo **AI chatbot** built using **Next.js + TypeScript**, streaming chat UI, and the **OpenRouter free models** API provider.  
This project showcases how to integrate a fully functional chat interface without requiring a paid OpenAI key.

---

## 📌 Features

- Chat UI with streaming responses
- Works with **free AI models** via **OpenRouter**
- Modern Next.js + React UI
- Ready to deploy on Vercel
- Handles approvals and tool states in chat UI

---

## 📥 Getting Started

### Prerequisites

Make sure you have the following installed:

- Node.js v18+
- npm (v9+)

---
